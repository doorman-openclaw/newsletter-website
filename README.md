# Newsletter Website

A landing page for a newsletter management application, inspired by [trydot.app](https://www.trydot.app).

## Features

- Clean, minimalist design
- Newsletter organization and categorization
- Read later functionality
- Smart filtering
- Privacy-focused messaging
- Simple pricing section
- Responsive layout

## Stack

- Pure HTML/CSS (no frameworks)
- Mobile-responsive
- Lightweight and fast

## Deploy

This site can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static host

## Preview

Open `index.html` in your browser to view locally.
